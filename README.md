## 【산수】 A와 B는 합쳐서 1100원, A는 B보다 1000원 비싸다. 그러면 B의 가격은?

```
A + B = 1100     --①
A - B = 1000     --②

A = 1000 + B     --②
①--
   (1000 + B) + B = 1100
               2B = 1100 - 1000
               2B = 100
                B = 50
```
